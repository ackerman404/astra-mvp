## E2E Flow Verification

### Flow 1: First Run Without API Key ✓
```
App starts → AstraApp.__init__() → get_api_key() returns None
→ _show_api_key_setup() dialog → Shows path + instructions
→ User sets up key → App proceeds to startup screen
```

### Flow 2: Normal Start with Ingestion ✓
```
Startup screen → "Ingest Documents" clicked → Progress bar appears
→ Background thread processes files → Progress updates shown
→ Completion dialog → "Start Session" enabled → Session screen
```

### Flow 3: Session with Layout Toggle ✓
```
Session screen → Click "⇕" toggle → Layout switches horizontal
→ QSplitter orientation changes → Panels rearrange
→ Click again → Layout switches back to vertical
```
