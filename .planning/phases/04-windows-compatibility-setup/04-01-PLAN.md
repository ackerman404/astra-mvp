Create new `audio_capture.py` module with:

1. **Abstract base class `AudioCapture`** with methods:
   - `start_capture()` - Begin capturing system audio
   - `stop_capture() -> np.ndarray` - Stop and return buffer
   - `get_last_n_seconds(n: int) -> np.ndarray` - Get recent audio
   - `get_audio_level() -> float` - Get RMS level for UI
   - `list_devices() -> list[dict]` - List available devices
   - `@property device` - Current device name

2. **`LinuxAudioCapture(AudioCapture)`** implementation:
   - Move `SystemAudioCapture` logic from transcriber.py
   - Uses `parec` subprocess for capture
   - Uses `pactl` for device listing

3. **Factory function `get_audio_capture(device=None)`**:
   - Detect platform using `sys.platform`
   - Return `LinuxAudioCapture` for Linux
   - Raise NotImplementedError for Windows (Plan 02 adds it)

4. **Move `list_monitor_devices()` and `list_audio_sources()`** from config.py to audio_capture.py (they're audio capture concerns, not config).

Keep same buffer size, sample rate handling. Don't change Whisper transcription logic.
