  **1. Create `LicenseActivationScreen(QWidget)` class** (place after `StartupScreen` class, before `AstraWindow`):

  Layout (vertical, centered, 400x450 min size, white background):
  - Title: "Astra Interview Copilot" (18pt bold, #222)
  - Subtitle: "License Activation" (12pt, #666)
  - Spacer
  - License key input: QLineEdit with placeholder "Enter your license key" (14pt, rounded border, centered text)
  - Status label: QLabel for feedback messages (12pt, initially hidden, word-wrap enabled)
  - Activate button: QPushButton "Activate" (blue #4a90d9, white text, 50px height, rounded, same style as startup buttons)
  - Spacer
  - Purchase link: QLabel with clickable HTML `<a href='#'>Where do I get a license key?</a>` — connect linkActivated signal to open URL in browser via `QDesktopServices.openUrl(QUrl("https://astra-copilot.com"))`. Style the link #4a90d9.
  - "Continue without license" link: QLabel with clickable text, smaller font (9pt, #999)

  Signals to emit:
  - `activated = pyqtSignal()` — emitted after successful activation
  - `skipped = pyqtSignal()` — emitted when user clicks "Continue without license"

  **Activate button handler (`_on_activate`)**:
  - Disable button, show "Activating..." in status label (color: #666)
  - Get key from input, strip whitespace
  - If empty: show "Please enter a license key" in red (#dc3545), re-enable button, return
  - Call backend activation (same logic as current `_show_license_key_setup`):
    - `requests.post(f"{base}/v1/license/activate", json={"license_key": key, "hardware_id": hw_id}, timeout=10)`
    - On 200: `save_license_key(key)`, show "License activated successfully!" in green (#28a745), emit `activated` signal after 500ms delay (QTimer.singleShot)
    - On non-200: parse error message from response JSON (`resp.json().get("detail", {}).get("error", {}).get("message", "Activation failed.")`), show in red (#dc3545)
      - Specifically for "already activated" / "in use" errors: show the server message directly
    - On `requests.ConnectionError`: `save_license_key(key)`, show "Key saved — will validate when online" in yellow/amber (#ffc107 background, #856404 text), emit `activated` after 1000ms
    - On other Exception: show f"Error: {e}" in red
  - Re-enable button after error states

  **Status label styling helper** (`_set_status(msg, color_type)`):
  - "success": green background (#d4edda), green text (#155724), green border (#c3e6cb)
  - "error": red background (#f8d7da), red text (#721c24), red border (#f5c6cb)
  - "warning": yellow background (#fff3cd), yellow text (#856404), yellow border (#ffeeba)
  - "info": gray background (#e2e3e5), gray text (#383d41)
  - Apply as rounded-corner label with padding (8px 12px, border-radius 6px, border 1px solid)

  **2. Modify `AstraApp.__init__`**:
  - Remove the `if not get_license_key(): self._show_license_key_setup()` block
  - Create `self.activation_screen = LicenseActivationScreen()`
  - Connect `self.activation_screen.activated` to `self._on_license_activated`
  - Connect `self.activation_screen.skipped` to `self._on_license_skipped`
  - In `__init__`, check `get_license_key()`:
    - If key exists: proceed normally (show startup screen)
    - If no key: show activation screen instead of startup screen

  **3. Add `_on_license_activated` method to AstraApp**:
  - Hide activation screen
  - Show startup screen

  **4. Add `_on_license_skipped` method to AstraApp**:
  - Hide activation screen
  - Show startup screen (user can ingest docs but Start Session will check license)

  **5. Modify `AstraApp._on_start_session`**:
  - Current behavior: shows QMessageBox warning if no license — change to:
  - If no license key: show the activation screen (hide startup screen, show `self.activation_screen`)
  - Re-connect activated signal to show session window after activation

  **6. Delete `_show_license_key_setup` method** — fully replaced by LicenseActivationScreen

  **7. Add import**: `from PyQt6.QtGui import QDesktopServices` (add to existing QGui import line), `from PyQt6.QtCore import QUrl` (add to existing QCore import line)
