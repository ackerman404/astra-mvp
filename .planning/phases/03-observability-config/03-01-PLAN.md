4. Add handler for model change:
   ```python
   def _on_model_changed(self):
       """Handle Whisper model selection."""
       model = self.model_combo.currentText()
       self.status_label.setText(f"Status: Loading {model}...")

       # Reload in background to avoid UI freeze
       def reload():
           from transcriber import reload_whisper_model
           reload_whisper_model(model)
           self.signals.status_update.emit("Status: Ready")

       thread = threading.Thread(target=reload, daemon=True)
       thread.start()
   ```
